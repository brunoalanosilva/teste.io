1 Perguntas

Explicar, com suas palavras, o que é e para que serve o algoritmo de MH e o gibbs sampling.

O algoritmo de Metropolis Hasting serve para amostrar valores de uma distribuição desconhecida, no contexto de bayesiana a distribuição desconhecida é a posteriori O algoritmo de gibbs sampling serve para gerar valores para posterioris com 2 ou mais variáveise utiliza a ideia de posterioris condicionais para gerar valores para cada parâmetro.

O algoritmo que eu utilizarei a seguir combina essas duas idéias, primeiro surgi a idéia de Gibbs Sampling em gerar os valores pelas posterioris condicionais, mas como é difícil gerar valor da posteriori condicional de \(\alpha\) é utlizado o algoritmo de Metropolis Hasting para gerar esses valores.

Obs: O algortimo utilizado foi em grande parte inspirado (ou copiado kk) do tutorial sugerido, a modificação que foi feita foi na parte, não ficou muito claro qual o nome do método que o autor utilizou para gerar o valor da posteriori condicional de \(\alpha\), apesar de haver muitas semelhanças com o algoritmo de Metropolis Hasting (falta o termo da distribuição proposta), logo incluímos esse termo fazendo o algoritmo ser uma mistura de de Gibbs sampling com metropolis Hasting.
